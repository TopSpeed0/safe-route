    🔍 איך זה עובד?

    📡 מקור הנתונים

    נתונים היסטוריים מ-dleshem/israel-alerts-data — ארכיון קוד פתוח של כל ההתרעות של פיקוד העורף מאז יולי 2014. מכיל 138,000+ אזעקות מעובדות (רקטות, כטב"מ, חדירות).

    אזעקות בזמן אמת מגיעות ישירות מ-פיקוד העורף דרך SSE (Server-Sent Events) — ללא עיכוב, ללא צד שלישי. הנתונים ההיסטוריים מתעדכנים כל 6 שעות.

    📊 מה מוצג?

    • אירועים = כל פעם שהופעלה אזעקה (ישירות מפיקוד העורף)
    • אזעקות = סה"כ ערים שנפגעו. אירוע אחד מפעיל אזעקה בכמה ערים בו-זמנית
    • ערים = כמה ערים ייחודיות קיבלו לפחות אזעקה אחת בתקופה
    • 🔴 Live = אזעקות בזמן אמת — מוצגות כ-toast + צליל ברגע שמתקבלות

    🗺️ המפה

    1,449 ערים מוצגות כפוליגונים צבעוניים (גבולות מוניציפליים). הצבע משקף את כמות האזעקות בתקופה שנבחרה — ירוק (מעט) עד אדום (הרבה).

    🛣️ חישוב מסלול

    כשבוחרים עיר מוצא ויעד, המערכת שולחת בקשה ל-OSRM — שרת ניתוב קוד פתוח — ומקבלת את המסלול האמיתי על הכבישים. אחר כך סורקת את כל 1,449 הערים ובודקת אילו נמצאות עד 2 ק"מ מהכביש.

    🎯 כיוון בליסטי

    מחשב את הכיוון האווירי (great circle bearing) מטהרן לכתובת שלך, ומראה איזה צד של הבניין חשוף לכיוון איראן ואיזה מוגן. כולל ניתוח קומה, חלון ראשי, וחלון ממ"ד.

    🔔 התראות חכמות

    מנתח דפוסי אזעקות לפי יום ושעה, מחשב הסתברויות, ומאפשר להגדיר סף להתראה אוטומטית עם צליל. כולל מצב "חכם" שמזהה מגמות (עלייה/ירידה/גל חדש).

    🏗️ ארכיטקטורה

    • Backend: Node.js API server (port 3850) — pure Node, אפס dependencies
    • Data: dleshem CSV (GitHub) + פיקוד העורף API (real-time)
    • Frontend: Vanilla JS + Leaflet + Chart.js — קובץ HTML יחיד
    • Live: SSE (Server-Sent Events) — heartbeat כל 5 שניות, אזעקות מיידיות

    ⚠️ הבהרה

    הכלי מבוסס על נתונים היסטוריים ובזמן אמת ומיועד לעזור בתכנון. הוא לא מחליף את הנחיות פיקוד העורף. תמיד יש לעקוב אחרי ההנחיות הרשמיות.

    ⚠️ תנאי שימוש

    אתר זה מספק כלי עזר לתכנון מסלול בלבד.

    • אינו מחליף את פיקוד העורף או כל גורם ביטחוני רשמי.
    • בדקו התראות בזמן אמת באפליקציית פיקוד העורף לפני כל נסיעה.
    • הנתונים מבוססים על מידע ציבורי ועלולים להכיל אי דיוקים.
    • האתר לא אחראי לכל נזק שייגרם כתוצאה משימוש באתר.
